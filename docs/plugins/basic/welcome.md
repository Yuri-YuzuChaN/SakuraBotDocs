# 入群欢迎

检测新入群人员，并给予欢迎词，可自定义并且支持图片

## 指令

指令前缀为 `gw`

- `gw help`：发送指令帮助
- `gw get`： 获取当前的欢迎词
- `gw list`：获取群内保存过的欢迎词
- `gw new 欢迎词`：设置一条新的欢迎词，允许携带图片，如果从未设置过欢迎词，将自动默认为新的欢迎词
- `gw set 序号`：设置已拥有的欢迎词，序号请使用指令 `gw list` 获取
- `gw del 序号`： 删除一条欢迎词，序号请使用指令 `gw list` 获取
